Kaos.Collections

Welcome to KaosCollections

KaosCollections consists of three primary classes:

  • RankedDictionaryTKey, TValue- SortedDictionaryTKey, TValue merged with SortedListTKey, TValue plus enhancements and improved performance.

  • RankedSetT- an enhanced version of SortedSetT.

  • RankedBagT- a multiset collection data structure.

All provided classes closely emulate the API of the Microsoft's classes while extending capability and improving performance in many common scenarios. All classes add list functionality so that any element can be accessed by index or the index of an element may be determined from the element's key.

This library is built as a .NET Standard project with multi-targeting. All targeted binaries are available in the distributed NuGet package. Current targets are:

  • .NETStandard 1.0. Use this for any .NET Core application or any .NET Framework application at or above .NET 4.5. This target does not include binary serialization support.

  • Full .NET Framework 4.0. This target does not include IReadOnlyDictionary or IReadOnlyCollection implementation.

  • Full .NET Framework 3.5. This target does not include IReadOnlyDictionary, IReadOnlyCollection implementation. It does not derive from ISet but does implement its members.

Off-line documentation is available as a MS-Help 1 (.chm) file. Also, installing as a NuGet package will provide IntelliSense and object browser documentation. On-line documentation is available at the link below.

Distribution

Complete source code and documentation is hosted at GitHub.com. Primary binary hosting is now at NuGet.org with downloads also available at GitHub.com. Complete documentation is available for download at GitHub.com as a .chm file.

Build environment

Visual Studio 2017 Community Edition (or greater) is required to build the library. There is an additional dependency on Sandcastle Help File Builder for building documentation. Test is performed by MSTest.
