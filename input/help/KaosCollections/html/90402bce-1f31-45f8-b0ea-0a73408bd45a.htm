Version History

Each section on this page describes a product release of KaosCollections.

History

Version 3.0.0 (2017-??-??)

  • Change root namespace from Kw to Kaos

  • Convert build to .Net Standard 1.0 multi-targetted to .NET 4.0 and .NET 3.5

  • Rename BtreeDictionary class to RankedDictionary with improved SortedDictionary emulation

  • Enhance RankedDictionary with SortedList emulation

  • Add RankedSet generic class which emulates SortedSet with order statistic capabilities

  • Add RankedBag generic class which provides multiset capabilities

Version 2.1.1 (2012-03-28)

  • Remove possible code coverage artifacts from .dll file

  • Initial release of documentation

Version 2.1.0 (2012-03-06)

  • Add support for range queries

  • Add optimized implementation of Last()
