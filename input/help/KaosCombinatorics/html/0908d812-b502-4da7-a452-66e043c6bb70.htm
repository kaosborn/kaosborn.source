Kaos.Combinatorics

Welcome to KaosCombinatorics

The Kaos.Combinatorics library contains four combinatorics classes and one static support class. See the namespace summary for details.

This library is built as a .NET Standard project with multi-targeting. All targeted binaries are available in the distributed NuGet package or as individual .dll files. Current targets are:

  • .NETStandard 1.0. Use this for any .NET Core application or any .NET Framework application at or above 4.5.

  • Full .NET Framework 4.0.

  • Full .NET Framework 3.5.

Distribution

Source and binary hosting was formerly at CodePlex.com as the KwCombinatorics project. Current hosting of full source code and documentation is at GitHub.com. Primary binary hosting is now at NuGet.org. This includes .xml files to support intellisense. Complete documentation is available for download at GitHub.com as a .chm file.

Build environment

This library requires Visual Studio 2017 Community Edition to build. There is an additional dependency on Sandcastle Help File Builder for building documentation. Test is performed by MSTest with code coverage of 100%.
