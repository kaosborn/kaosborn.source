PermutationGetRows Method

Iterate thru all rows of the Permutation table for every value of Rank ascending.

Namespace:  Kaos.Combinatorics
Assembly:  KaosCombinatorics (in KaosCombinatorics.dll) Version: 5.0.0
Syntax
C#

public IEnumerable<Permutation> GetRows()

Return Value

Type: IEnumerablePermutation
An iterator for a Permutation table.
Remarks
If the start row is not of Rank 0, the iteration will wrap around so that RowCount items are always produced.

using System;
using Kaos.Combinatorics;

namespace ExampleApp

    static class PnExample01

        static void Main()
        {
            var pn = new Permutation (choices: 4);

            Console.WriteLine ("n={0}:\n", pn.Choices);

            foreach (var row in pn.GetRows())
                Console.WriteLine ("{0,2}:  {1}", row.Rank, row);
        }
