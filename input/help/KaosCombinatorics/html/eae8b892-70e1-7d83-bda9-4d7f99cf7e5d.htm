MulticombinationGetRows Method

Iterate thru all rows of the Multicombination table for every value of Rank ascending.

public IEnumerable<Multicombination> GetRows()

Return Value

Type: IEnumerableMulticombination
An iterator for a Multicombination table.
Remarks
If the start row is not of Rank 0, the iteration will wrap around so that RowCount items are always produced.
Examples
C#
Copy
using System;
using Kaos.Combinatorics;

namespace ExampleApp
{
    static class McExample01
    {
        static void Main()
        {
            var mc = new Multicombination (choices:4, picks:3);

            Console.WriteLine ("n={0}, k={1}:\n", mc.Choices, mc.Picks);

            foreach (var row in mc.GetRows())
                Console.WriteLine ("{0,2}:  {1}", row.Rank, row);
        }

        /* Output:

        n=4, k=3:

         0:  { 0, 0, 0 }
         1:  { 0, 0, 1 }
         2:  { 0, 0, 2 }
         3:  { 0, 0, 3 }
         4:  { 0, 1, 1 }
         5:  { 0, 1, 2 }
         6:  { 0, 1, 3 }
         7:  { 0, 2, 2 }
         8:  { 0, 2, 3 }
         9:  { 0, 3, 3 }
        10:  { 1, 1, 1 }
        11:  { 1, 1, 2 }
        12:  { 1, 1, 3 }
        13:  { 1, 2, 2 }
        14:  { 1, 2, 3 }
        15:  { 1, 3, 3 }
        16:  { 2, 2, 2 }
        17:  { 2, 2, 3 }
        18:  { 2, 3, 3 }
        19:  { 3, 3, 3 }

        */
    }
}
